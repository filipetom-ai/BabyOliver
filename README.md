# BabyOliver

App web do bebê em português do Brasil para acompanhar:

- mamadas
- sono
- fraldas
- bomba
- remédios
- anotações

## Estrutura

- `docs/`: versão publicada no GitHub Pages
- `baby-tracker-app/`: fonte local do MVP
- `baby-tracker-concepts/`: conceitos visuais iniciais

## Publicação

Depois do push para o GitHub:

1. Abra o repositório.
2. Vá em `Settings`.
3. Abra `Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch `main`.
6. Selecione a pasta `/docs`.
7. Salve.

O GitHub vai gerar um link público para o app.

## Observação

Esta primeira versão usa armazenamento local do navegador e backup por arquivo JSON.
Para sincronização automática entre você e sua esposa, o próximo passo será conectar um backend.
